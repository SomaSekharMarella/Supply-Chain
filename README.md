# 🚜 AgriTrace – Blockchain-Based Agriculture Supply Chain

AgriTrace is a decentralized agriculture supply chain tracking system built using blockchain technology.  
It ensures transparency, traceability, and trust across the entire farm-to-consumer journey.

# ⛏️ Proof of Work Based Crowdfunding Platform (Project Simera)

A decentralized crowdfunding platform built on blockchain with a **Proof-of-Work governance mechanism**.

## 🚀 Key Features

- Smart contract-based fund management
- Proof-of-Work validation before withdrawal
- Community-based governance
- Transparent on-chain transactions
- Secure and tamper-proof architecture

## 🛠️ Tech Stack

- **Blockchain:** Ethereum (Sepolia Testnet)
- **Smart Contracts:** Solidity
- **Development Framework:** Hardhat
- **Frontend:** React + Vite
- **Backend:** Spring Boot
- **Database:** MySQL
- **Storage:** IPFS
